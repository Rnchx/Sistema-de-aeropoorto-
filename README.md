# ✈️ Sistema de Aeroporto

Projeto desenvolvido com o objetivo de simular e gerenciar operações de um aeroporto, permitindo o controle de voos, passageiros, aeronaves e demais funcionalidades relacionadas ao ambiente aeroportuário.

## 📌 Sobre o Projeto

O **Sistema de Aeroporto** foi criado para facilitar o gerenciamento de informações aeroportuárias de forma prática e organizada.  
O sistema possui uma interface intuitiva e funcionalidades voltadas para administração de voos e controle operacional.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Node.js
- Express
- MongoDB *(caso utilize banco de dados)*

## 🎯 Funcionalidades

- ✅ Cadastro de voos
- ✅ Gerenciamento de passageiros
- ✅ Controle de aeronaves
- ✅ Sistema de login
- ✅ Painel administrativo
- ✅ Consulta de informações de voos
- ✅ Organização de dados aeroportuários

## ⚙️ Como Executar o Projeto

### 🔹 Clone o repositório

```bash
git clone https://github.com/Rnchx/Sistema-de-aeropoorto-.git
```

### 🔹 Acesse a pasta do projeto

```bash
cd Sistema-de-aeropoorto-
```

### 🔹 Instale as dependências

```bash
npm install
```

### 🔹 Execute o projeto

```bash
npm start
```
